# Digital Flashcard Learner

## Problem
Students often struggle to effectively memorize information for exams, lacking a personalized and organized method for self-testing. Traditional physical flashcards are cumbersome to manage and don't offer dynamic learning features or progress tracking.

## Solution
This website allows users to create custom sets of digital flashcards for any subject. It offers a practice mode using spaced repetition principles and tracks user progress locally, providing an efficient and clutter-free way to master new material.

## Features
*   **Custom Flashcard Sets**: Create, edit, and delete your own flashcard sets.
*   **Spaced Repetition Practice**: Study mode designed to optimize memorization using spaced repetition principles.
*   **Local Progress Tracking**: Monitor your learning progress and performance for each set.
*   **Modern UI**: Stunning dark theme with vibrant accents and glassmorphic elements.

## Getting Started

To run the development server:

1.  **Clone the repository:**
    ```bash
    git clone [repository-url]
    cd flashcard-learner-app
    ```
2.  **Install dependencies:**
    ```bash
    npm install
    ```
3.  **Run the development server:**
    ```bash
    npm run dev
    ```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

The app will automatically reload when you make changes.

## How to Use

1.  **Home**: Get an overview of the application.
2.  **My Sets**: Create new flashcard sets, or manage existing ones.
3.  **Study Session**: Select a set and begin your practice session.
4.  **Progress**: View your learning statistics and performance over time.

Enjoy mastering your material!
